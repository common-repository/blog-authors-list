A Plugin to display blog authors list on page using shortcode.

== Description ==

This is plugin to show the list of blog authors on wordpress page using shortcode.

This plugin will display the blog author name , number of posts posted by author , author bio info and link to view all the posts posted by author. 

Tutorial link: http://www.php-mastermind.com/display-list-blog-authors-number-posts-wordpress-page/

== Installation ==

1. Upload `pm-blog-authors-list` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_shortcode('[blog_authors_list]'); ?>` in your templates or place the [blog_authors_list] shortcode in your posts/page content to display the list of blog authors
